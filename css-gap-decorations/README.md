# CSS Gap Decorations demos

This directory contains demos that showcase the use of [CSS Gap Decorations](https://github.com/MicrosoftEdge/MSEdgeExplainers/blob/main/CSSGapDecorations/explainer.md), a feature by the Microsoft Edge team to draw line decorations within gaps in CSS Grid, Flexbox, and Multi-column layouts.

## Try the feature

CSS Gap Decorations is available by default in Microsoft Edge 149 and above (or another Chromium-based browser with a matching version). Just open any of the demos above in a supported browser.
