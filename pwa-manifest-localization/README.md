# Manifest Localization for Web Apps

➡️ **[Open the demo](https://microsoftedge.github.io/Demos/pwa-manifest-localization/)** ⬅️

This demo showcases the Web App Manifest `x_localized` members feature, which allows Progressive Web Apps to provide localized app metadata (such as name, description, and icons) for different languages directly in the manifest file.

The demo supports English, German, and Arabic localizations.

## How to use

1. Open the [demo site](https://microsoftedge.github.io/Demos/pwa-manifest-localization/) in a supported browser.
2. Install the PWA in one of the supported languages (English, German, or Arabic). The PWA will display localized values.
3. To see the localization's app update flow in action, try switching your browser's Preferred Language settings to another supported language. On relaunch, you will be prompted to update the installed PWA to the new localized values.
