# Scoped Custom Element Registries

➡️ **[Open the demo](https://microsoftedge.github.io/Demos/scoped-custom-element-registries/)** ⬅️

Scoped Custom Element Registries allow you to define custom elements that are scoped to a specific shadow roots, elements, or documents, rather than being globally available.

To learn more, see:

* [CustomElementRegistry](https://developer.mozilla.org/en-US/docs/Web/API/CustomElementRegistry) at MDN.
