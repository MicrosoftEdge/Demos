OpaqueRange demo

The OpaqueRange API lets you create ranges over the value of <textarea> and <input> elements, enabling getBoundingClientRect() and the CSS Custom Highlight API to work directly on form controls without cloning elements.

⚠️ OpaqueRange is not supported in this browser. OpaqueRange is available starting in Chromium 147 (Edge 147). If your browser is older, enable it in edge://flags or chrome://flags by searching for "OpaqueRange".

Use case 3: Live range tracking

OpaqueRange is live — it automatically updates its offsets as the content changes via interactive edits (typing, backspace, delete) or setRangeText(). Click Highlight "hello" to highlight the word, then type before it and watch the highlight follow.

Use case 4: Disconnecting an OpaqueRange

Calling range.disconnect() detaches the range from its element. The offsets reset to 0, geometry becomes empty, and the range stops receiving live updates. Create a range, then disconnect it and observe the change.

Use case 5: Multi-line getClientRects()

OpaqueRange.getClientRects() returns one rectangle per line for multi-line selections. Select a range that spans multiple lines and see each individual line rect.
